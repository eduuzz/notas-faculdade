4. Importacao via Portal Auto (fluxo principal)

Este e o fluxo mais complexo do sistema. Envolve os 3 servicos trabalhando juntos.

Por que Playwright? Requisicoes HTTP diretas (axios/fetch) ao portal retornam "Usuario sem permissao". O portal TOTVS valida que as requisicoes vem do AngularJS com contexto de sessao completo. So funciona navegando de verdade.

5. Operacoes do Dia-a-Dia

Editar notas:
    SistemaNotas --> atualizarDisciplina(id, { ga: 8.5, gb: 7.0 })
    --> supabase.update().eq('id', id).eq('user_id', user.id)

Iniciar disciplina:
    --> atualizarDisciplina(id, { status: 'EM_CURSO', semestreCursado: '2026/1' })

Remover disciplina:
    --> supabase.delete().eq('id', id).eq('user_id', user.id)

Adicionar manualmente:
    --> adicionarDisciplina({ nome, periodo, ... })
    --> supabase.insert([...])
